# Smart-_Park-_System# Smart Park — Real-Time Parking Operations Monitoring System

Smart Park is a browser-based dashboard built to monitor parking occupancy, analyze usage trends, and support better parking operations through real-time data visualization.

It was created as a portfolio project to showcase operational analysis, real-time monitoring, and process improvement using HTML, CSS, and JavaScript.

## Live Demo

🔗 [View Live on GitHub Pages](https://priyasesh.github.io/Smart_Park_System/)

## What It Does

This project simulates a smart parking monitoring system that helps track parking space occupancy and identify operational patterns.

It helps users:
- Monitor total, occupied, available, reserved, and maintenance spaces
- View parking spot status across multiple zones
- Track occupancy trends over time
- Detect high-usage areas and underutilized zones
- View operational alerts based on parking demand
- Generate simple insights to support decision-making
- Export the current parking snapshot as a CSV file
- Switch between light and dark mode

## Features

### KPI Summary Cards
Displays key parking metrics such as total spots, occupied spots, available spots, occupancy rate, reserved spaces, and maintenance spaces.

### Parking Space Map
Shows a live visual layout of parking spots with color-coded status indicators.

### Search and Filters
Allows users to filter by zone, status, and slot ID or vehicle type.

### Live Simulation
Simulates real-time parking occupancy changes to reflect a live monitoring environment.

### Hourly Occupancy Trend
Shows how parking occupancy changes across the day to help identify peak times.

### Zone Utilization Analysis
Compares parking usage across different zones to identify overload and underuse.

### Operational Alerts
Displays capacity-related alerts when occupancy becomes high.

### Insights Panel
Summarizes current parking conditions with operational observations.

### Event Feed
Shows recent parking events such as occupancy changes and maintenance updates.

### CSV Export
Exports the filtered parking data for reporting or review.

### Theme Toggle
Supports both light and dark mode.

It improves visibility into parking usage, highlights inefficiencies in space utilization, and supports faster operational decision-making during peak demand periods.
